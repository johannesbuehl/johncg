# Text
Playout a lower-thirds with custom text.

## Properties

### Text
Custom text to display in the lower thirds.

The text wraps automatically, individual linebreaks are also supported.

## CasparCG layers

### template
`JohnCG/Text`

#### JSON-data
```json
{
	"text": "Lorem Ipsum"
}
```
